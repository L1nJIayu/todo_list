# todo_list
TODO列表，React练习项目

## 功能清单
- [x] 添加新事项
- [x] 删除新事项
- [x] 修改未完成的事项内容,可修改描述内容
- [x] 切换按钮:用于显示详情的switch按钮
- [x] 状态更新按钮:点击后,3秒,状态由[未完成]转换为[已完成]
- [x] 查看已完成历史记录, 可检索,可删除,不可修改
